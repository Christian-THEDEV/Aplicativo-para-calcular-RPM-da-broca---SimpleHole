# Aplicativo-para-calcular-RPM-da-broca---SimpleHole

Versão: 1.0

O SimpleHole é um aplicativo desenvolvido para o Windows com o objetivo de auxiliar na furação de materiais utilizando uma furadeira de bancada. O Aplicativo em formato de console traz praticidade em seu uso, cujo há um menu para escolher o material e exigindo além disso, só o diâmetro da broca, onde o aluno ou o funcionário só precisará de medir a broca com um paquímetro ou usando o número que está no corpo da broca e colocar no sistema, facilitando assim a vida da pessoa para evitar que decore fórmulas para tal.

Caso utilize o código fonte para a melhoria desse programa, dê-me os créditos porfavor :D.
